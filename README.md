# Aviothic.ai - Breast Cancer Detection AI Landing Page

A modern, responsive landing page for Aviothic.ai, a healthcare AI startup focused on revolutionizing breast cancer screening.

## Features

- Responsive design using Tailwind CSS
- Animated elements with Framer Motion
- Modern UI components
- Contact form
- Information about the startup, team, and technology

## Technologies Used

- React.js
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and visit `http://localhost:5173`
